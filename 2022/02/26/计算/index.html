计算

程序要求：1、有地方放输入的数字
2、有办法输入数字
3、输入的数字能参与计算


输入：输入也在终端框框口中
输入是以行位单位进行的，行的结束标志就是你按下了回车键。在你按下回车之前，你的程序不会得到任何东西

变量

■int price=0;

■这一行，定义了一个变量。变量的名字是price，类型是int，初始值是0。

■变量是一个保存数据的地方，当我们需要在程序里保存数据时，比如上面的例子中要记录用户输入的价格，就需要一个变量来保存它。用一个变量保存了数据，它才能参加到后面的计算中，比如计算找零。


变量定义

■变量定义的一般形式就是：

<类型名称><变量名称>; 如int amount; int price,amount;

■变量的名字

变量需要一个名字，变量的名字是一种“标识符”，意思是它是用来识别这个和那个的不同的名字。

标识符有标识符的构造规则。基本的原则是：标识符只能由字母、数字和下划线组成，数字不可以出现在第一个位置上，C语言的关键字（有的地方叫他们保留字）不可以用做标识符。

赋值和初始化

■int price=0;

■这一行，定义了一个变量。变量的名字就是price，类型是int，初始值是0。

■price=0是一个式子，这里的“=”是一个赋值运算符，表示将“=”右边的值赋给左边的变量。

赋值：

和数学中不同，a=b在数学中表示a和b的值相等（交换律），而在程序设计中则表示一个动作，即：b→a（将b的值赋给a），关系是静态的，而动作是动态的。

■当赋值发生在定义变量的时候，程序的结果和预期就会有出路。

■** 要求scanf这个函数读入下一个整数，读到的结果赋值给变量price**

** ■小心“&”（传值符）**


定义一个常量，比如100元找零，想要定义100元这个量，可以用 “const int amount=100”

■const是一个修饰符，加在int的前面，用来给这个变量加上一个const（不变的）的属性。这个const的属性表示这个变量的值一旦初始化，就不能再做修改了

■如果你试图对常量做修改，把它放在赋值运算符的左边，就会被编译器发现，指出为一个错误。


例题：美国人固执地使用英制计量单位，他们习惯用几尺几寸（英尺英寸）来报他们的身高。如果遇到一个美国人告诉你他是5英尺7，他的身高应该是一米几呢？

错解：

#include <stdio.h>

int main( ){

int foot,inch;

printf(“请分别输入身高的英尺和英寸：”“如输入“5，7” 表示5英尺7英寸”);

scanf(“%d %d”,&foot,&inch);

int high;

high=(foot+inch/12)*0.3048;

printf(“身高是%d m”,high);


return 0;

}

正解1（推荐）：#include <stdio.h>

int main( ){

float foot,inch;

printf(“请分别输入身高的英尺和英寸：”“如输入“5，7” 表示5英尺7英寸”);

scanf(“%f %f”,&foot,&inch);

float high;

high=(foot+inch/12)*0.3048;

printf(“身高是%f m”,high);


return 0;

}

正解2（不太推荐）

讲程序中的12改成12.0

思考：为什么用第一种写法输出的结果是错误的呢？为什么输入5尺7寸和5尺2寸（3寸4寸）得到的结果是一样的呢？

答：因为两个整数的运算的结果只能是整数，用int表示变量是整型，题中的7/12明显是一个大于零小于1的小数（零点几几）因此进行取整之后就是0，因此结果输出会出现问题

一个表达式是一系列运算符和算子的组合，用来计算一个值

如：count=count+1;

■ 运算符（operator）是指进行运算的动作，比如加法运算符“+”，减法运算符“-”，等号“=”，取余运算符“%d”等

■ 算子(operand) 是指参与运算的值，这个值可能是常数，也可能是变量，还可能是一个方法的返回值

例题：输入两个时间，每个时间分别输入小时和分钟的值，然后输出两个时间之间的差，也以几小时几分表示

解：

#include <stdio.h>

int main ( ) {

int hour1,hour2;

int minute1,minute2;

int t1,t2,t;

printf(“请输入末时间点的小时位和分钟位：”);

scanf(“%d %d”,&hour2,&minute2);

printf(“请输入初时间点的小时位还分钟位：”);

scanf(“%d %d”,&hour1,&minute2);

t1=hour1*60+minute1;

t2=hour2*60+minute2;

t=t2-t1;

printf(“时间差为%d小时%d分钟”,t/60,t%60);

return 0;

}

什么叫单目、双目、三目？

单目：一个式子中只有一个运算数如 i++ i– (表示一个数的正负如-2)

双目：一个式子中有两个运算数如a+b a-b a*b

三目：一个式子中有三个运算数 a？b：c

“+” “-”作为给数定义正负的时候单目不变，此时是单目运算符（从右向左），优先级比乘除还要高，作为运算符的时候是是双目运算符

递增递减运算符

“++” “–”是两个很特殊的运算符，它们是单目运算符，这个算子还必须是变量。这两个运算符分别叫做递增和递减运算符，他们的作用就是给这个变量+1或-1。

count++；

count+=1；

count=count+1；

三者等价


前缀后缀

++和–可以放在变量的前面，叫做前缀形式，也可以放在变量的后面，叫做后缀形式。

a++的值是a加1以前的值，而++a的值是加了1以后的值，无论哪个，a自己的值都加1了

怎么理解：

#include<stdio.h>

int main (){

int a;

a=10;

printf(“a++=%d\n”,a++);

printf(“%d\n”,a);

printf(“++a=%d\n”,++a);

printf(“%d\n”,a);

return 0;

}

运行结果：a++=10; //a的初始值是10，a++表示+1以前的值，故结果为10
